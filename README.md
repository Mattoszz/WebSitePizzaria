# WebSitePizzaria
 
 Criei um website para uma pizzaria, o qual atende o cliente no formato desktop como no formato para dispositivos móveis.
 
 ### Qual a expectativa do projeto?
 Este website atende perfeitamente a uma pizzaria, com promoções, depoimentos, produtos e mais.
 
 ### Foi desenvolvido com
 Este projeto foi desenvolvido com a linguagem de marcação de hipertexto HTML, Foi usado o CSS para estilizar e fazer algumas animações com o animate, também foi usado a linguagem de programação JavaScript para fazer as interações e algumas animações com o JQuery. 
 Foi usado a IDE Visual Studio Code.
 
 ### Local para baixar o código:
 O projeto está WebSitePizzaria, na pasta projeto pizza zmattos
